隐私政策

适用产品：谐音梗（移动端应用）
生效日期：2026-04-16

1. 我们是谁

本隐私政策适用于“谐音梗”（下称“本应用”）。我们非常重视你的个人信息与隐私保护，并将依据本政策说明我们如何收集、使用、共享与保护你的信息。

2. 我们收集哪些信息

2.1 你主动提供的信息

  • 本应用当前不要求你注册账号，也不要求你提交真实身份信息。
  • 当你使用“系统分享”功能时，你选择分享的内容将由系统分享面板处理。

2.2 设备与使用信息（可能由第三方 SDK 收集）

为了实现广告展示、反作弊、安全保障与统计分析等目的，第三方 SDK 可能会收集与设备/使用有关的信息，例如：

  • 设备标识符（如 IDFA/AAID、设备型号、系统版本、语言与地区）。
  • 网络信息（如 IP 地址、网络类型、运营商信息）。
  • 应用运行信息（如崩溃信息、性能数据、广告曝光/点击等事件）。

提示：iOS 设备的广告标识符（IDFA）获取通常需要你在系统弹窗中授权。未授权时，广告 SDK 可能使用其他合规方式进行投放与频控。

2.3 本地存储信息

为了提升体验，本应用会在本地保存部分数据（卸载应用后会被系统清除），包括但不限于：

  • 答题进度（如当前题目页码）。
  • 提示积分与提示规则展示状态。
  • 图片缓存文件（用于加速图片显示、减少重复下载）。

3. 我们如何使用信息

  • 提供并维持核心功能：题目获取、图片展示、答题交互与设置页功能。
  • 性能优化：使用图片缓存减少网络请求与加载时间。
  • 广告展示与奖励逻辑：用于展示开屏/激励视频广告，并在完成条件后发放提示积分。
  • 安全与合规：用于防作弊、反滥用、风险控制与故障排查。

4. 第三方 SDK 与共享

本应用可能集成以下类型的第三方服务（实际以你安装的版本为准）：

  • 广告服务：Google Mobile Ads（AdMob），友盟 U-APPWIN（UMUnionSDK）。
  • 图片加载与缓存：用于下载/缓存题图（例如 CachedNetworkImage、CacheManager）。

第三方 SDK 可能会依据其隐私政策独立处理数据。我们建议你查阅对应第三方的隐私政策以了解其数据处理方式。

5. 权限说明

  • 网络权限：用于请求题库数据与加载题图、拉取广告。
  • 跟踪权限（iOS 可能出现）：用于获取 IDFA 以提升广告投放与衡量效果；你可在系统中拒绝。

6. 你的权利

  • 你可以在系统设置中管理广告相关权限（例如 iOS 的跟踪权限）。
  • 你可以在本应用“清除缓存”功能中清理图片缓存。
  • 你可以卸载本应用以清除本地存储数据。

7. 信息安全

我们会采取合理的技术与管理措施保护你的信息安全，防止信息被未经授权访问、披露、使用、修改或毁坏。 但请理解，任何互联网传输或存储方式都无法保证绝对安全。

8. 未成年人保护

若你是未成年人，请在监护人指导下使用本应用。我们建议监护人帮助未成年人阅读本政策并做出合适选择。

9. 本政策更新

我们可能适时更新本隐私政策。更新后我们会在适当位置提示你查看最新版本。若你继续使用本应用，即表示你同意更新后的政策。
